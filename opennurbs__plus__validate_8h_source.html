84 /* Find a local minimum of a one parameter function using a quadratic approximation algorithm.
85  * This algorithm does not use derivative information.
86  *
87  * INPUT:
88  * ax, bx, cx a bracketed minimum satisfying conditions 1 and 2.
89  * 1) ax < bx < cx.
90  * 2) f(bx) < f(ax) and f(bx) < f(cx).
91  * farg
92  * pointer passed to function f()
93  * f
94  * evaluation function with prototype
95  * int f(void* farg,double t,double* ft)
96  * f(farg,t,&ft) should compute ft = value of function at t
97  * -1: failure
98  * 0: success
99  * 1: |f(x)| is small enough - will return *t_addr = x
100  * and the return code 1.
101  * desired_min - once a function value this small is found, stop.
102  * stop_tol - once it is determined that a min exists in an interval of this size, stop
103  * max_it ( >= 2)
104  * maximum number of iterations to permit (when in doubt use 100)
105  *
106  * OUTPUT:
107  * *t_addr abcissa of a local minimum between ax and cx.
108  * ON_SimpleQuadraticMinimize()
109  * 0: successful
110  * 1: The search for a minimum was stopped because the function f()
111  * returned a 1 indicating that the current value of t was small
112  * enough.
113  * 2: After max_iteration_cnt iterations the tolerance restrictions
114  * where not satisfied. Try increasing max_it,
115  * or use the value of (*t_addr) with extreme caution.
116  * -1: failure
117  *
118  *
119  */
120 ON_DECL
121 int ON_SimpleQuadraticMinimize(int (*f)(void*, double, double*), void* f_arg,
122  double ax, double bx, double cx,
123  double desired_min,
124  double stop_tol,
125  int max_it,
126  double *t_addr=0,
127  double *min_f=0

130 /*
131 Class to check the relationship between pairs of curves in a set.
132 Looks for pairs that overlap.
133  Only consider overlaps that start and end at curve endpoints.
134  Check 5 points including endpoints
135  Classify overlaps as
136  match - curves seem to be identical to within close_g0_tol
137  partial - overlap starts at an endpt of one curve and ends at an endpt of the other.
138  parts of each curve are not contained in the overlap
139  supercurve, subcurve - one curve is contained within the other.
140 Looks for endpoints that are close but not close enough, wrt g0, g1, g2;
141  If endpt A is within close_gx_tol of endtpt B, but not within good_gx_tol, it is reported.
142 g0 tol is 3d distance.
143 g1 tol is angle in radians
144 g2 tol is a percentage - if fabs(kA - kB) < tol/200*(kA + kB), then within tol
145 */

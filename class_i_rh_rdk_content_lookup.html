Looks up a content in the relevant location (typically a document). IMPORTANT **** This will also find non-document contents that are globally registered. If this global lookup is not desired, please call LookUpContentInDocument() instead.

Parameters
uuidInstanceis the instance id of the content to look up.
uuidLocationis an optional location identifier **** This is currently not used.
Returns
A pointer to the content if found, else null.

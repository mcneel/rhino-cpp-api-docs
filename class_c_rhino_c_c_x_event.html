enum CRhinoCCXEvent::EVENT_TYPE

There are two valid curve-curve intersection events CCX_POINT - curves intersect in an isolated point a[0] = a[1] = first curve parameter A[0] = A[1] = intersection point on first curve b[0] = b[1] = second curve parmeter B[0] = B[1] = intersection point on second curve

CCX_OVERLAP (a[0],a[1]) = first curve parameter range A[0] = intersection start point on first curve A[1] = intersection end point on first curve (b[0],b[1]) = second curve parameter range B[0] = intersection start point on second curve B[1] = intersection end point on second curve

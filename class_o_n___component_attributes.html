◆ EdgeFlags

enum ON_ComponentAttributes::EdgeFlags : unsigned int

ON_EdgeAttributeFlags are used to report attributes of single edge objects, like ON_SubDEdge and ON_BrepEdge, and aggregate edge demographics in objects with topology like ON_SubD, ON_Mesh and ON_Brep.

See also
ON_BrepEdge::ComponentAttributes, ON_SubDEdge::ComponentAttributes, ON_Brep::AggregateEdgeComponentAttributes, ON_Mesh::AggregateEdgeComponentAttributes, ON_SubD::AggregateEdgeComponentAttributes

summary> The Open bit is set when an edge has distinct start and end vertices. /summary> summary> The Closed bit is set when an edge begins and ends at the same vertex. /summary>

summary> The Boundary bit is set when an edge has one face. /summary>

summary> The Interior bit is set when an edge has two distinct faces. /summary>

summary> The Nonmanifold bit is set when an edge has three or more faces. /summary>

Nonmanifold 

summary> The Oriented bit is set when an edge has two faces with compatible orientations. /summary>

Oriented 

summary> The NotOriented bit is set when an edge has two faces with opposited orientations. /summary>

NotOriented 

summary> The Smooth bit is set when an an edge has two faces with a guaranteed surface tangent continuity. /summary>

Smooth 

summary> The Crease bit is set when an edge has two faces with a possible surface tangent discontinuity /summary>

Crease 

summary> The Dart bit is set when an edge has two faces with a possible surface tangent discontinuity at one end and guaranteed surface tangent space continuity at the other end. /summary>

Dart 

summary> The Seam bit is set when an edge has two faces that are identical and the edge is on the parametric boundary of the face's surface. /summary> example> Parametric surfaces that are cylinders are an example of a situation where seam edges occur. /example>

Seam 

summary> The Slit bit is set when edge has two faces that are identical and the edges is not a seam. /summary>

Slit 

summary> The Slit bit is set when an edge has zero length. /summary>

Degenerate 

summary> The Damaged bit is set when an edge has a critical flaw like missing vertex information. /summary>

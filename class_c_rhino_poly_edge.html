enum CRhinoPolyEdge::eTanMode

Description: The CRhinoPolyEdge EvSrf... tools can evaulate the "binormal" surface tangent direction in several different ways. The binormal surface tangent always points "out". @untitled table tmAuto if trim is a srf iso, then tm_IsoDir, else tmPerp tmIsoDir srf tangent will be parallel to du or dv tmPerp srf tangent will be perpindicular to edge See Also: CRhinoPolyEdge::SrfTangentMode CRhinoPolyEdge::SetSrfTangentMode

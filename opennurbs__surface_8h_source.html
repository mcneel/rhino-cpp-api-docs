448  /*
449  Description:
450  Search for a derivatitive, tangent, or curvature
451  discontinuity.
452  Parameters:
453  dir - [in] If 0, then "u" parameter is checked. If 1, then
454  the "v" parameter is checked.
455  c - [in] type of continity to test for.
456  t0 - [in] Search begins at t0. If there is a discontinuity
457  at t0, it will be ignored. This makes it
458  possible to repeatedly call GetNextDiscontinuity
459  and step through the discontinuities.
460  t1 - [in] (t0 != t1) If there is a discontinuity at t1 is
461  will be ingored unless c is a locus discontinuity
462  type and t1 is at the start or end of the curve.
463  t - [out] if a discontinuity is found, then *t reports the
464  parameter at the discontinuity.
465  hint - [in/out] if GetNextDiscontinuity will be called
466  repeatedly, passing a "hint" with initial value *hint=0
467  will increase the speed of the search.
468  dtype - [out] if not nullptr, *dtype reports the kind of
469  discontinuity found at *t. A value of 1 means the first
470  derivative or unit tangent was discontinuous. A value
471  of 2 means the second derivative or curvature was
472  discontinuous. A value of 0 means teh curve is not
473  closed, a locus discontinuity test was applied, and
474  t1 is at the start of end of the curve.
475  cos_angle_tolerance - [in] default = cos(1 degree) Used only
476  when c is ON::continuity::G1_continuous or ON::continuity::G2_continuous. If the
477  cosine of the angle between two tangent vectors is
478  <= cos_angle_tolerance, then a G1 discontinuity is reported.
479  curvature_tolerance - [in] (default = ON_SQRT_EPSILON) Used
480  only when c is ON::continuity::G2_continuous. If K0 and K1 are
481  curvatures evaluated from above and below and
482  |K0 - K1| > curvature_tolerance, then a curvature
483  discontinuity is reported.
484  Returns:
485  Parametric continuity tests c = (C0_continuous, ..., G2_continuous):
486 
487  true if a parametric discontinuity was found strictly
488  between t0 and t1. Note well that all curves are
489  parametrically continuous at the ends of their domains.
490 
491  Locus continuity tests c = (C0_locus_continuous, ...,G2_locus_continuous):
492 
493  true if a locus discontinuity was found strictly between
494  t0 and t1 or at t1 is the at the end of a curve.
495  Note well that all open curves (IsClosed()=false) are locus
496  discontinuous at the ends of their domains. All closed
497  curves (IsClosed()=true) are at least C0_locus_continuous at
498  the ends of their domains.
499  */

507  int* hint=nullptr,

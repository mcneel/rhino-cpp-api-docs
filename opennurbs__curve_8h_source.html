462  /*
463  Description:
464  Search for a derivatitive, tangent, or curvature
465  discontinuity.
466  Parameters:
467  c - [in] type of continity to test for.
468  t0 - [in] Search begins at t0. If there is a discontinuity
469  at t0, it will be ignored. This makes it
470  possible to repeatedly call GetNextDiscontinuity
471  and step through the discontinuities.
472  t1 - [in] (t0 != t1) If there is a discontinuity at t1 is
473  will be ingored unless c is a locus discontinuity
474  type and t1 is at the start or end of the curve.
475  t - [out] if a discontinuity is found, then *t reports the
476  parameter at the discontinuity.
477  hint - [in/out] if GetNextDiscontinuity will be called
478  repeatedly, passing a "hint" with initial value *hint=0
479  will increase the speed of the search.
480  dtype - [out] if not nullptr, *dtype reports the kind of
481  discontinuity found at *t. A value of 1 means the first
482  derivative or unit tangent was discontinuous. A value
483  of 2 means the second derivative or curvature was
484  discontinuous. A value of 0 means teh curve is not
485  closed, a locus discontinuity test was applied, and
486  t1 is at the start of end of the curve.
487  If 'c', the type of continuity to test for
488  is ON::continuity::Gsmooth_continuous and the curvature changes
489  from curved to 0 or 0 to curved and there is no
490  tangency kink dtype is returns 3
491  cos_angle_tolerance - [in] default = cos(1 degree) Used only
492  when c is ON::continuity::G1_continuous or ON::continuity::G2_continuous. If the
493  cosine of the angle between two tangent vectors is
494  <= cos_angle_tolerance, then a G1 discontinuity is reported.
495  curvature_tolerance - [in] (default = ON_SQRT_EPSILON) Used
496  only when c is ON::continuity::G2_continuous. If K0 and K1 are
497  curvatures evaluated from above and below and
498  |K0 - K1| > curvature_tolerance, then a curvature
499  discontinuity is reported.
500  Returns:
501  Parametric continuity tests c = (C0_continuous, ..., G2_continuous):
502 
503  true if a parametric discontinuity was found strictly
504  between t0 and t1. Note well that all curves are
505  parametrically continuous at the ends of their domains.
506 
507  Locus continuity tests c = (C0_locus_continuous, ...,G2_locus_continuous):
508 
509  true if a locus discontinuity was found strictly between
510  t0 and t1 or at t1 is the at the end of a curve.
511  Note well that all open curves (IsClosed()=false) are locus
512  discontinuous at the ends of their domains. All closed
513  curves (IsClosed()=true) are at least C0_locus_continuous at
514  the ends of their domains.
515  */

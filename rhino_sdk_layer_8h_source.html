23  /*
24  Description:
25  Puts the layer tree information in the m_layer_index,
26  m_parent_layer_index, and m_child_layer_index[] array fields.
27  Parameters:
28  layer_index - [in]
29  index of "this" layer
30  list_parents - [in]
31  0 = no,
32  1 = immediate parent
33  If this layer has a parent, then m_parent_count=1 and
34  the returned m_parent_list[] array has one element.
35  2 = immediate parent, their parent, and so on.
36  If this layer has a parent, then m_parent_count>=1 and
37  the returned m_parent_list[] is sorted by generation.
38  m_parent_list[0] = immediate parent,
39  m_parent_list[1] = grandparent,
40  m_parent_list[2] = great grandparent,
41  and so on.
42  list_children - [in]
43  0 = no,
44  1 = immediate children
45  If the layer has children, then m_child_count is
46  the number of immediate children the returned
47  m_child_list[] array list is sorted by layer index.
48  2 = immediate children, their children, and so on.
49  If the layer has children, then m_child_count is
50  the number of children, grandchildren, and so on,
51  and the returned m_child_list[] array list is
52  sorted by layer index.
53  bIgnoreDeletedLayers - [in]
54  If true (the default), deleted layers are not added to the lists.
55  If false, deleted layers are included.
56  Returns:
57  True if input and layer table are valid and false if an index is out
58  of range or an error was detected in the layer tree information.
59  */
60  bool Create(
61  class CRhinoLayerTable& layer_table,
62  int layer_index,
63  int list_parents,
64  int list_children,
65  bool bIgnoreDeletedLayers = true
66  );
67 
68  void Destroy();

675  /*
676  Description:
677  Search for a layer with a matching name.
678  Parameters:
679  layer_name - [in]
680  bSearchDefaultLayers - [in]
681  If true, the system default layers (DefaultLayer) are included in the search.
682  If you are searching for a layer to use as it exists,
683  bSearchDefaultLayers should be true.
684  If you are searching for a layer to modify, bSearchDefaultLayers
685  should be false because the default layers cannot be modified.
686  bSearchFullPathNames - [in]
687  true:
688  search for the full path name parent_name::...:leaf_name that uniquely identify a layer.
689  false:
690  search layer names (when the layer tree contains child nodes, there may be
691  multiple layers with the same name.
692  not_found_return_value - [in]
693  Value to return if no layer has a matching name.
694  Depending on what should happen if the name is not found,
695  CurrentLayerIndex(), DefaultLayer.Index() and ON_UNSET_INT_INDEX
696  are common choices for this value.
697  multiple_found_return_value - [in]
698  Value to return if bSearchFullPathNames is false and two or more layers have a matching name.
699  Depending on what should happen if the name is not found,
700  CurrentLayerIndex(), DefaultLayer.Index() and ON_UNSET_INT_INDEX
701  are common choices for this value.
702  Returns:
703  The index of the matching layer is returned.
704  If no match is found, then not_found_return_value is returned.
705  If multiple matches are found, then multiple_match_return_value is returned.
706  Example:
707  If bSearchFullPathNames is falues and there are layers with indices and names:
708 
709  0: "alpha"
710  1: "beta"
711  2: "beta::alpha"
712  3: "beta::gamma"
713  4: "beta::gamma::pi"
714  5: "beta::gamma::omega"
715  5: "beta::gamma::omega::pi"
716 
717  Then
718  FindLayerFromUniqueName( "alpha" ) returns 0.
719  FindLayerFromUniqueName( "beta::alpha" ) returns 2.

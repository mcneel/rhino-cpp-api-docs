8 /** \class CRhRdkVariant
9 
10  The RDK variant is an object that can hold data of different types. It is primarily
11  used by the automatic user interface system to provide a unified way of passing
12  parameters to and from the user interface.
13 
14  A null variant has a type of vtNull and no value.
15 
16  Use the 'varying' feature to represent multiple different values of the
17  same type. This is useful for property pages etc. If you use this feature,
18  you \e must check Varies() before retrieving the value. If you never call
19  SetVaries() then you never have to call Varies().
20 
21  When reading the value, conversion will be performed if necessary.
22 
23  \note It is not possible to retrieve the value if the variant
24  is null or varying. An assert will occur if this is attempted.
25 
26  This class has no virtual functions and should not be subclassed.
27 */

Detailed Description

This class represents a 'sun'. It is used to get a light that represents a sun, but please note that this is not necessarily the RDK Document Sun. To use it, create one on the stack. Call Sun() to get an interface to the sun. Call the interface's various set methods to configure the sun the way you want it. Then call Light() to get an ON_Light to use as the sun.

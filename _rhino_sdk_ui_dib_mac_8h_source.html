37 // Class to manage DIB's. This class will create a DIB, a HBITMAP and associated memory
38 // DC. The HBITMAP and CRhinoDib::FindBits() point to the smae memory. You can modify the
39 // bits directly or use GDI with the associated memory DC to modify the bitmap. The HBITMAP
40 // does not get selected by the memory devidce context due to the fact that the DIB can
41 // only be selected in one device context at a time. To select or remove the Bitmap() from
42 // its memory DC call DCSelectBitmap( bool). If you call DCSelectBitmap( true) you may
43 // cast a CRhinoDib as a CDC* or HDC and use it directly.

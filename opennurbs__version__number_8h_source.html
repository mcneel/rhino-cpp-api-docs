21 Description:
22  Create a 4-byte unsigned integer value that has desireable version

24 
25 Parameters:
26  major_version - [in]
27  major_version >= 0 and major_version <= 63
28

30  minor_version >= 0 and minor_version <= 127

32  year - [in]
33  year >= 2000 and year <= 2099
34 
35  month - [in]
36  month >= 1 and month <= 12
37 
38  day_of_month - [in]
39  day_of_month >= 1 and day_of_month <= ON_DaysInMonthOfGregorianYear(year,month)
40 
41  platform_branch - [in]
42  0: developer build
43  1: Windows Commercial build
44  2: Mac Commercial build
45  3: Windows BETA build
46  4: Mac Beta build
47  5: Windows WIP build
48  6: Mac WIP build
49 
50 Returns:
51  If the input values are valid, this returns
52  a 4-byte nonzero version number with the following properties:
53  verA = ON_VersionNumberConstruct(majorA,minorA,<timeA>,branchA)
54  verB = ON_VersionNumberConstruct(majorB,minorB,<timeB>,branchB)
55  - If majorA > majorB, then verA > verB.
56  - If majorA = majorB and minorA > minorB, then verA > verB.
57  - If majorA = majorB and minorA = minorB and
58  timeA > timeB, then verA > verB.
59  - If majorA = majorB, minorA = minorB, timeA = timeB, and
60  branchA > branchB, then verA > verB.
61  If any input is not valid, then zero is returned.
62 */

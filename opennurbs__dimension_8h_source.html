557  /*
558  Description:
559  The angle between the lines is dimensioned.
560 
561  If the lines intersect in a single point, that point is used as the center
562  of the angular dimension arc. In this case, there are eight possible angles
563  to dimension. The point_on_angular_dimension_arc and point_on_line parameters
564  are used to select the correct angle to dimension. If a point_on_line parameter
565  is not set, the corresponding line's midpoint is used.
566 
567  If the lines are colinear, the point on the line closest to
568  point_on_angular_dimension_arc is the center of the angular dimension arc.
569 
570  Parameters:
571  dim_style - [in]
572  Pass nullptr if a dim_style is not available.
573  line1 - [in]
574  point_on_line1 - [in]
575  If point_on_line1 is specified, it inidicates which semi-infinite portion of line1 to dimension.
576  Otherwise the midpoint of lne1 as a segment is used.
577  When in doubt, pass ON_3dPoint::UnsetPoint.
578  line2 - [in]
579  point_on_line2 - [in]
580  If point_on_line2 is specified, it inidicates which semi-infinite portion of line2 to dimension.
581  Otherwise the midpoint of line2 as a segment is used.
582  When in doubt, pass ON_3dPoint::UnsetPoint.
583  point_on_angular_dimension_arc - [in]
584  A point on the interior of the angular dimension arc.
585  bSetExtensionPoints - [in]
586  If bSetExtensionPoints is true, and a point_on_line parameter is valid, that point
587  is used as the extension point. Otherwise the angular dimension arc endpoint is used.
588  Returns:
589  True if successful.
590  False if input is not valid. In this case ON_DimAngle::Empty settings are returned.
591  */

NOTE: The m_stretch_function and m_stretch_factor are for expert users only. It is prefered to use the GetStretchFactor and SetStretchFactor functions instead.

The m_stretch_function is a monotone increasing function f() such that: f(0) = 0, f(1) = 1, f'(0) = 0, f'(1) = 1

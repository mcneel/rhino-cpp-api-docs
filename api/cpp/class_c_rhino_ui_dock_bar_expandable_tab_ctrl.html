bool CRhinoUiDockBarExpandableTabCtrl::CreateExpandableTabCtrl ( CRhinoUiDockBar *  pParent)

Description: Create window to hold the tab control. This window will size itself to fit the entire client area of the dock bar it is embedded in. If window is created succesfully then a tab control will be created and embeded in this window. Parameters: pParent [in] Dock bar to embed window in. Returns: Returns true if this window and the tab control associated with it are created successfully.

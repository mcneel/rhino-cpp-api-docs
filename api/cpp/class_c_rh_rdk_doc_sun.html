Detailed Description

This class is obsolete and should not be used. Please use CRhRdkDocument::Sun() or CRhRdkDocument::Skylight() instead. For example: 

CRhinoDoc& doc = ...

CRhRdkDocSun sun(doc);
if (sun.Sun().EnableOn())
    ...
if (sun.SkylightOn())
    ...

The above is obsolete; change to: 

    auto& rdkDoc = CRhRdkDocument::Get(doc);
    if (rdkDoc.Sun().EnableOn())
        ...
    if (rdkDoc.Skylight().On())
        ...

Detailed Description

Description: Component indices are used to provide a persistent way to identify portions of complex objects.

Member Enumeration Documentation

◆ TYPE

enum ON_COMPONENT_INDEX::TYPE

Do not change these values; they are stored in 3dm archives and provide a persistent way to indentify components of complex objects.

Detailed Description

Command instance specific history recording support. Description: This allows the global history recording option to be optionally turned on or off for this single instance of the command execution. When command execution terminates the inital setting of History Recording is restored. The option is implemented by a command line toggle option. The option is allowed if and only if command specific history recording is enabled.

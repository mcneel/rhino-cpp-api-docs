123 Description:

126 Parameters:
127  args - [in] The sweep1 arguments
128  shape_blending - [in] Shape blending, where:
129  Local (shape_blending = 0), the sweep stays constant at the

131  Global (shape_blending = 1), the sweep is linearly blended from

143 /*
144 Description:
145  Make a 1-Rail or 2-Rail sweep with one or two rail curves and one or more shape curves
146 Parameters:
147  output_surfaces - [out] Array that will return sweep results
148  pRail - [in] Curve to use for sweep rail
149  shapes - [in] Array of curves to use for sweep shapes
150  Shape curves must be going the correct direction
151  They will be sorted for position along the rail.
152  bClosed - [in] If the rail curve is closed this determines if the
153  sweep will be closed or will use only part of the rail.
154  If there are more than one shape and bClosed is false
155  the sweep will go only between the first and last
156  shape as sorted along the rail.
157  If there is one shape and bClosed is false, the sweep
158  will go from the shape to the end of the rail.
159  tolerance - [in] Tolerance used by the sweep function for fitting
160  the surface.
161 Returns:
162  true - success
163  false - error
164 */

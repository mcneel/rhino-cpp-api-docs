Description: Commands that should be localized by whose names should not autocomplete should be derived from CRhinoHiddenCommand. These are typically used when modeless UI changes the document and the change needs to be undoable by the undo command. Paremeters: bTransparent - [in] true if command is transparent and can be run inside of other commands. bDoNotRepeat - [in] true if command should not be repeated. pPlugIn - [in] if command is part of a plug-in, the plug-in pointer must be passed here. (This is automatically handled by the plug-in wizard.) Remarks: All commands derived from CRhinoTestCommand are "test" commands and are not available in the commercial release.

bool CArgsRhinoGetCurve::SetSubDFriendly ( bool  bSubDFriendly)

Parameters: bSubDFriendly - [in] Returns: Resulting value of SubDFriendly(). Remarks: SubD friendly curves are cubic, nonrational, uniform NURBS that are either periodic or have zero 2nd derivatives at the start and end. If SubDFriendly() is true, then degree, knot spacing, and end condition paramaters are ignored in curve creation commands that use CArgsRhinoGetCurve for input.

bool CArgsRhinoGetCurve::SubDFriendly ( ) const

Returns: True if the resulting curve will be a cubic, uniform, nonrational and either periodic or have zero 2nd deriviatives at the ends. Remarks: SubD friendly curves are cubic, nonrational, uniform NURBS that are either periodic or have zero 2nd derivatives at the start and end.

17 /** \class IRhRdkContentUI
18 
19  This interface represents the user interface (UI) of a content. A content's UI appears inside
20  an editor which is represented by IRhRdkContentEditor. The UI is the part of the editor which
21  allows the user to view and modify the persistent state of the selected content(s).
22 
23  This interface is implemented by RDK internally in the core content class, CRhRdkCoreContent.
24  If you do not override CRhRdkCoreContent::CreateUI() you do not have to concern yourself with
25  the details. If however, you do want to override CreateUI() to create your own custom UI, then
26  you must derive your UI class from CRhRdkCustomContentUI and implement this interface to support
27  your own UI design. \note CRhRdkCustomContentUI implements some of this interface for you.

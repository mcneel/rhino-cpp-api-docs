bool ON_FileIterator::Initialize ( const wchar_t *  directory_name)

Iteratation initialization tools Description: Initialize where the search should occur. Parameters: directory_name - [in] The directory to look in. item_name_filter - [in] If this paramter is null, then the iteration includes all names in the directory. The item name to search for. This parameter can include wildcard characters, such as an asterisk (*) or a question mark (?). For example, "\rootdir\subdir\*.*" will iterate all files in the \rootdir\subdir\ directory.

Returns: true: The iterator is set to the first item. false: There are no matching items.

Remarks: Calling FirstItem() is eqivalent to calling Initialize() and then calling NextItem().

1034  // If you need to make any additions or changes to
1035  // these enum values, then please read comment in
1036  // CRhinoFileReadOptions::CRhinoFileReadOptions().
1037 
1038  // These mode names are confusing and misleading, but
1039  // they cannot be changed or the SDK will break.
1040  // Read the comments carefully. When in doubt,
1041  // read everything and put it in the model.

1044  // true means we are merging whatever is being

1046  // you need to consider things like:
1047  // * If the information being read is in a different
1048  // unit system, it should be scaled if UseScaleGeometry
1049  // is true.
1050  // * There can be existing layers, fonts, materials,
1051  // dimension styles, hatch patterns, and so on with the
1052  // same name as items being read from the file.

1054

1056  // true means we are reading the information into an empty

1058  // * Setting the unit system (if the file has a unit system)

1060  // * Setting up appropriate views when you're finished reading.

1062

1069  // true means we are reading information that will be used to
1070  // create an instance definition or some other type of
1071  // "inserting" that is supported by Rhino's "Insert" command.
1072  InsertMode,
1073 
1074  // ImportReferenceMode: (default = false)

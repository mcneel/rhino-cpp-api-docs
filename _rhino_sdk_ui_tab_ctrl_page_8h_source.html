50  // Description:
51  // This method is called by CRhinoUiTabPageCtrl when the user chooses the
52  // OK or the Apply Now button. When CRhinoUiTabPageCtrl calls this method,
53  // changes made on all tab control pages in the CRhinoUiTabPageCtrl are
54  // accepted, the tab control page retains focus, and OnApply returns true.
55  // Before OnApply can be called by the framework, you must have called

62  // should return true to accept changes and false to prevent changes

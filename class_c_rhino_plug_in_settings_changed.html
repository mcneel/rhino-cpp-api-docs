virtual void CRhinoPlugInSettingsChanged::Notify ( const class CRhinoPlugInSettingsChanged::CParameters &  params)
pure virtual

Rhino will call Notify() CRhinoPlugIn::SaveSettings() is called or when a plug-in's settings are saved while Rhino is closing. This event is useful when you want to keep plug-in settings synchronized with multiple Rhino sessions. Rhino will only notify you if the plug-in Id passed to the constructor is the plug-in writing the persistent settings file.

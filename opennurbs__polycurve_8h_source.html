26 /*
27  Description:
28  An ON_PolyCurve is an ON_Curve represented by a sequence of
29  contiguous ON_Curve segments. A valid polycurve is represented
30  by an array m_segment of Count()>=1 curve objects and a strictly
31  increasing array m_t of Count()+1 parameter values. The i-th
32  curve segment, when considered as part of the polycurve, is affinely
33  reparamaterized from m_t[i] to m_t[i+1], i.e., m_segment[i].Domain()[0]
34  is mapped to m_t[i] and m_segment[i].Domain()[1] is mapped to m_t[i+1].
35 */

256  /*
257  Description:
258  Search for a derivatitive, tangent, or curvature discontinuity.
259  Parameters:
260  c - [in] type of continity to test for. If ON::continuity::C1_continuous
261  t0 - [in] search begins at t0
262  t1 - [in] (t0 < t1) search ends at t1
263  t - [out] if a discontinuity is found, the *t reports the
264  parameter at the discontinuity.
265  hint - [in/out] if GetNextDiscontinuity will be called repeatedly,
266  passing a "hint" with initial value *hint=0 will increase the speed
267  of the search.
268  dtype - [out] if not nullptr, *dtype reports the kind of discontinuity
269  found at *t. A value of 1 means the first derivative or unit tangent
270  was discontinuous. A value of 2 means the second derivative or
271  curvature was discontinuous.
272  cos_angle_tolerance - [in] default = cos(1 degree) Used only when
273  c is ON::continuity::G1_continuous or ON::continuity::G2_continuous. If the cosine
274  of the angle between two tangent vectors
275  is <= cos_angle_tolerance, then a G1 discontinuity is reported.
276  curvature_tolerance - [in] (default = ON_SQRT_EPSILON) Used only when
277  c is ON::continuity::G2_continuous or ON::continuity::Gsmooth_continuous.
278  ON::continuity::G2_continuous:
279  If K0 and K1 are curvatures evaluated
280  from above and below and |K0 - K1| > curvature_tolerance,
281  then a curvature discontinuity is reported.
282  ON::continuity::Gsmooth_continuous:
283  If K0 and K1 are curvatures evaluated from above and below
284  and the angle between K0 and K1 is at least twice angle tolerance
285  or ||K0| - |K1|| > (max(|K0|,|K1|) > curvature_tolerance,
286  then a curvature discontinuity is reported.
287  Returns:
288  true if a discontinuity was found on the interior of the interval (t0,t1).
289  Remarks:
290  Overrides ON_Curve::GetNextDiscontinuity.
291  */
292  bool GetNextDiscontinuity(
293  ON::continuity c,
294  double t0,
295  double t1,
296  double* t,
297  int* hint=nullptr,
298  int* dtype=nullptr,
299  double cos_angle_tolerance=ON_DEFAULT_ANGLE_TOLERANCE_COSINE,
300  double curvature_tolerance=ON_SQRT_EPSILON
301  ) const override;

303  /*
304  Description:
305  Test continuity at a curve parameter value.
306  Parameters:
307  c - [in] continuity to test for
308  t - [in] parameter to test
309  hint - [in] evaluation hint
310  point_tolerance - [in] if the distance between two points is
311  greater than point_tolerance, then the curve is not C0.
312  d1_tolerance - [in] if the difference between two first derivatives is
313  greater than d1_tolerance, then the curve is not C1.
314  d2_tolerance - [in] if the difference between two second derivatives is
315  greater than d2_tolerance, then the curve is not C2.
316  cos_angle_tolerance - [in] default = cos(1 degree) Used only when
317  c is ON::continuity::G1_continuous or ON::continuity::G2_continuous. If the cosine
318  of the angle between two tangent vectors
319  is <= cos_angle_tolerance, then a G1 discontinuity is reported.
320  curvature_tolerance - [in] (default = ON_SQRT_EPSILON) Used only when
321  c is ON::continuity::G2_continuous or ON::continuity::Gsmooth_continuous.
322  ON::continuity::G2_continuous:
323  If K0 and K1 are curvatures evaluated
324  from above and below and |K0 - K1| > curvature_tolerance,
325  then a curvature discontinuity is reported.
326  ON::continuity::Gsmooth_continuous:
327  If K0 and K1 are curvatures evaluated from above and below
328  and the angle between K0 and K1 is at least twice angle tolerance

331  Returns:
332  true if the curve has at least the c type continuity at the parameter t.
333  Remarks:
334  Overrides ON_Curve::IsContinuous.
335  */

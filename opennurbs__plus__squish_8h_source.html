144  /*
145  If -1.0 <= m_absolute_limit < 0.0, then then an absolute
146  compression limit is applied so that
147  (2d length)/(3d length) >= fabs(m_absolute_limit).
148  In particular, if m_absolute_limit = -1.0, then no compression
149  is permitted (2d length) >= (3d length).
150 
151  If 0.0 < m_absolute_limit <= 1.0 then then an absolute
152  stretching limit is applied so that
153  (2d length)/(3d length) <= 1/fabs(m_absolute_limit).
154 
155  Examples:
156  m_absolute_limit
157  1.0: no stretching, (2d length) <= 1.0*(3d length)
158  0.5: cap on stretching, 0.5*(2d length) <= (3d length)
159  -0.5: cap on compression, (2d length) >= 0.5*(3d length)
160  -1.0: no compression, (2d length) >= 1.0*(3d length)
161  */
162  double m_absolute_limit;
163

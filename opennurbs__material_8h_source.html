101  is a good default
102  N - [in]

104  R - [in]
105  3d reflection direction
106  Returns:
107  1.0:
108  The input values were not valid or the calculation failed due to
109  a divide by zero or some other numerical arithmetic failure.
110  fresnel reflection coefficient
111  1/2 * ((g-c)/(g+c))^2 * (1 + ( (c*(g+c) -1)/(c*(g+c) + 1) )^2)
112  where
113  c = N o (N-R); // c = 3d vector dot product of N and (N-R)
114  and
115  g = sqrt(fresnel_index_of_refraction*fresnel_index_of_refraction + c*c - 1.0).
116  */

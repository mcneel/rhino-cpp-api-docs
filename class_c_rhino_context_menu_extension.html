virtual void CRhinoContextMenuExtension::ExtendContextMenu ( CRhinoContextMenuContext &  context,
CRhinoContextMenu &  pop_menu 
)
pure virtual

Description: Required virtual override, this method will only be called if ShouldExtendContextMenu() returns true. This is called by Rhino after a separator has been added to the specifed context menu and is the plug-ins opportunity to add items to the context menu. Parameters: context [in] Identifies the context menu to extend and what items are currently selected pop_menu [in] Reference to the context menu object which will display the context menu, call the AddItem() mehtods on this object to extend the menu

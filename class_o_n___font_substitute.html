◆ Identification

enum ON_FontSubstitute::Identification : unsigned char
strong

MissingFontIdentification enumerates the ways a missing font can be specified. The method that is used determines on the context where the font is referenced. A ON_FontSubstitute class can support multiple ways.

Enumerator
Unset 

Unset value.

FontCharacteristics 

Font characteristics hash used to identify the missing font.

PostScriptName 

PostScript name used to identify the missing font. On Apple platforms the PostScript name typically identifies a unique face. On Windows platforms a group of simulated fonts, like Bahnschrift, can have the same PostScript name.

QuartetName 

Font face quartet (Windows LOGFONT) name used to identify the missing font. A font face quartet has up to four faces (regular, bold, italic, bold-italic). The absolute font weight cannot be used to reliably determine if a quartet face is bold. For example, Arial Black has a very high weight but is the regular face in its quartet.

QuartetFace 

Quartet name (Windows LOGFONT) name and face (regular, bold, italic, bold-italic) used to identify the missing font. The absolute font weight cannot be used to reliably determine if a quartet face is bold. For example, Arial Black has a very high weight but is the regular face in its quartet.

FamilyName 

Family name used to identify the missing font. A family can contain many faces with different weights, stretches, and styles.

FamilyFace 

Family face (name, weight, stretch, and style) used to identify the missing font.

const ON_Font *  substitute_font 
)
static

Description: Create from a quartet face (Windows LOGFONT) description. A quartet is a group of four faces (regular, bold, italic, bold-italic). On Windows this quartet is called a logical font or LOGFONT. There is no Apple OS analogue. Every quartet has a regular face but is is common for light and heavy fonts to not have bold members. Parameters: missing_font_quartet_name - [in] Quartet name. On Windows this is the Windows LOGFONT name. This name is guessed at on Apple platforms. missing_font_quartet_face_is_bold - [in] true if this is the bold or bold-italic member in the quartet. false if this is the regular or italic member in the quartet. missing_font_quartet_face_is_italic - [in] true if this is the italic or bold-italic member in the quartet. false if this is the regular or bold member in the quartet. substitute_font - [in] This must be an installed font from ON_Font::InstalledFontList()

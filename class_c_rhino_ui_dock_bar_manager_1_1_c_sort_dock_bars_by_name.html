Detailed Description

Description: This function is provided to simplify sorting by name. To use pass this function to CRhinoUiDockBarManager::Sort as the compar parameter and pass a CSortDockBarsByName pointer as p.

Example: RhinoUiDockBarManager().Sort( CRhinoUiDockBarManager::SortDockBarsByName, NULL, &CRhinoUiDockBarManager::CSortDockBarsByName() ); ON_SimpleArray<CRhinoUiDockBar*>sorted_list; RhinoUiDockBarManager().GetSortedList( sorted_list);

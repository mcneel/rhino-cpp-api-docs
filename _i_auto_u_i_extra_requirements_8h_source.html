6 /** \class IAutoUIExtraRequirements
7 
8  Extra requirements are a way of specifying extra functionality on parameters in the automatic UI.
9  All objects that support such extra UI requirements must implement this interface.
10 
11  RDK currently supports the following extra requirements:
12 
13  - RDK_TEXTURING Provides texturing controls; an 'on' checkbox, an 'amount' edit box, and a 'subnode' control.
14  - RDK_EXCLUDE_LIST Provides support for exclude lists (lists of things to be excluded from some operation).
15  - RDK_ENABLE_DISABLE Provides an 'on' checkbox for enable/disable of the control set.
16  - RDK_FIXED_DECIMAL_PLACES Allows customization of number of decimal places displayed in a numeric edit box.
17  - RDK_NUMBER_EDIT_TWEAKS Allows tweaking of numeric edit boxes.
18  - RDK_COLOR_BUTTON_TWEAKS Allows tweaking of color buttons.
19  - RDK_IOR_MENU Requests a context menu containing the indices of refraction for all kinds of substances.
20  - RDK_COMBOBOX
21  - RDK_DATETIME_LOCATION Requests a callback for providing location information to the date/time control.
22  - RDK_LOCATION_DATETIME Requests a callback for providing date/time information to the location control.
23  - RDK_CONTROLSET_ENABLE Requests a general enable/disable callback.
24 
25 */

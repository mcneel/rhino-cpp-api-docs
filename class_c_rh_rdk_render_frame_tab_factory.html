Detailed Description

RDK provides two tabs on the Render Window's 'effects' panel, Exposure and Post-Effects. This class is an RDK extension which allows users to add their own tabs to this panel.

Extended version only used on the Mac at the moment. Might be used on Windows in Rhino 7.

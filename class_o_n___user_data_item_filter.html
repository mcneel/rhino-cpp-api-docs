◆ m_item_id

ON_UUID ON_UserDataItemFilter::m_item_id

The item id for object user data is the value of ON_UserData.m_userdata_uuid. The item id for user table is the application id. A nil item id indicates the setting is applied to all object user data and user table information for the specified application.

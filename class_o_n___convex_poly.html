Detailed Description

This is a base class for a convex polytope in 3d space, i.e. the convex hull of a finite set of points called verticies.

This is the base type in the implementation of the GJK algorithm ClosestPoint(ON_ConvexPoly& A, ON_ConvexPoly& B, ...)
